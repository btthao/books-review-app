# Book reviews app

### Features:

- Add books to database
- Edit/delete books you created
- Rate books
- Bookmark your favorites
- User authentication, cookie

### Built with:

- GraphQL
- Apollo client, apollo server express
- PostgreSQL
- TypeORM
- Redis
- React
- TypeScript
- Next.js
- TypeGraphQL
- Express
- Docker
- Chart.js
- TailwindCSS
- Formik
