# Book reviews app

### Features:

- Add books to database
- Edit/delete books you created
- Rate books
- Bookmark your favorites
- User authentication, cookie

### Built with:

- GraphQL
- Apollo client, apollo server express
- PostgresSQL
- TypeORM
- Redis
- React
- TypeScript
- Next.js
- TypeGraphQL
- Node.js, Express
- TailwindCSS
- Formik
